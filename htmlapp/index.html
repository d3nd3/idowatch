IDO / VeryFit / Ryze based smartwatch activity downloader

This makes a bluetooth connection to IDO based smart watches and creates a TCX file from the activity data. This specifically doesn't mark the activity as sync'd to allow VeryFit/Ryze to continue operating

Requirements

  • Browser with bluetooth support : Currently only Chrome.
  • Activity hasn't be sync'd with VeryFit / Ryze app already

Limitations

  • Doesn't mark activities as sync'd, so you need to sync each activity with the app
  • Only supports running
  • Only supports a very limit protocol subset - will likely break
  • Doesn't support getting GPS path
  • May do unexpected things
